The name of a CodeCheck Configuration such as "AUTOSAR" or "My Configuration". This can also be the path for an existing CodeCheck configuration json file. The following result files will be output in the current workspace directory:
  • und_cc.sarif
  • CodeCheckResultByTable.csv
  • CodeCheckResultByFile.txt
  • CodeCheckResultByViolation.txt
To integrate with the Warnings Next Generation plugin, use the Sarif parser with und_cc.sarif as the pattern.
